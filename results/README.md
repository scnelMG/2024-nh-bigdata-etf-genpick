# Results

이 폴더는 GitHub 포트폴리오에 공개할 수 있는 파생 결과물을 정리한 공간입니다.

## Files

| File | Description |
| --- | --- |
| `clustering_model_scores.csv` | 군집화 알고리즘별 평가 결과 |
| `cluster_feature_importance.csv` | XGBoost 기반 군집 설명 Feature Importance |
| `cluster_metric_means.csv` | 군집별 ETF 지표 평균 |
| `cluster_coordinates.csv` | t-SNE 기반 군집 좌표 |
| `cluster_assignments_and_metrics.csv` | ETF별 군집 번호와 주요 지표 |
| `keyword_importance_shap.csv` | 텍스트 기반 수익률 모델의 전체 SHAP 키워드 중요도 |
| `sample_*_shap_values.csv` | 대표 종목별 SHAP 세부 값 샘플 |
| `sample_top_holdings_descriptions.csv` | 구성 비율 상위 종목 설명 입력 예시 |

원본 NH투자증권 제공 테이블은 데이터 권한과 용량 문제로 공개 대상에서 제외했습니다.
